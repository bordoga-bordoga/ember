## About The Project

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm

  ```sh
  npm install npm@latest -g
  ```

* PostgreSQL

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/AnotherBrynAlt/Ember.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```
